Miner
=

Aim to provide a Dispatcher framework. See test source for API usage.


How to add the dependency?
=

* In gradle

```groovy

   repositories {
       mavenCentral()
       maven {
           url "http://metaphor.github.io/maven-repo/releases/"
       }
   }

   dependencies {
       compile "metaphor:miner:3.0.0"
   }

```
