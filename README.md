Miner
=

Aim to provide a Pattern Matching framework.


How to add the dependency?
=

* In gradle

```groovy

   repositories {
       mavenCentral()
       maven {
           url "http://metaphor.github.io/maven-repo/releases/"
       }
   }

   dependencies {
       compile "metaphor:miner:1.1.0"
   }

```
